## 🚀 How to Run

### **Backend API:**
```bash
cd deployment/api
python app.py
# Access at http://localhost:8000
```

### **Streamlit Frontend:**
```bash
cd deployment/frontend
streamlit run streamlit_app.py
# Access at http://localhost:8501
```

### **React Frontend:**
```bash
cd deployment/frontend/react-app
npm install
npm run dev
# Access at http://localhost:3000
```
