## 🎓 Terminology Glossary

**Medical AI Terms:**
- CLAHE: Contrast Limited Adaptive Histogram Equalization
- Sensitivity/Recall: % of fractures correctly detected
- Specificity: % of normal X-rays correctly identified
- False Negative: Missed fracture (most critical metric)
- Grad-CAM: Gradient-weighted Class Activation Mapping
- DICOM: Digital Imaging and Communications in Medicine

**ML/AI Terms:**
- Transfer Learning: Using pre-trained model weights
- Fine-tuning: Training specific layers
- Ensemble: Combining multiple models
- Augmentation: Creating data variations
- Prefetching: Loading next batch while GPU trains

**LLM Terms:**
- Prompt Engineering: Crafting effective LLM instructions
- Token: Unit of text (~4 characters)
- Temperature: Randomness control (0=deterministic)
- Few-shot: Learning from examples in prompt

**Deployment Terms:**
- REST API: Web service using HTTP
- Async/Await: Non-blocking code execution
- CORS: Cross-Origin Resource Sharing
- Prometheus: Metrics collection system
- HIPAA: Healthcare data privacy law
