```
Integrated Gradients: IG(x) = (x - x') * ∫[0,1] ∂F(x' + α(x - x'))/∂x dα
Focal Loss: FL(p) = -α(1-p)^γ log(p)
PSI: Σ (current% - baseline%) * ln(current% / baseline%)
KS Test: D = sup|F₁(x) - F₂(x)|
```
